Details

For my compiler theory and programming language design class I took, we were tasked with implementing several components to a custom programming class that democracy named Kickflip. It was dynamically typed (against my wishes) and imperative, as well as compiled.

I developed this with my good friend Austin, and we implemented custom logic to handle tokens, as well as a Lexer class and a Parser. In the end, we had a working compiler for our custom langauge.

This project taught me a lot about how compilers work, and ultimately made me realise that I enjoyed programming language design and theory.
